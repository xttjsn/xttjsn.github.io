Secure Multiparty Computation

GASH

GASH is a programming framework for secure multi-party computation. It’s built on the theory of Yao’s Garbled Circuit.

GASH is designed for compiling circuit description files (written in gcdf language, which has somewhat similar syntax with C++) to binary circuits (in .circ format, which has similar syntax with the And Inverter Graph-AIG circuit syntax).

Then, GASH garbles the circuit described by the circ file (if the machine is garbler), connects to peer, and perform Yao’s protocol.

GASH Viewer

To minimize the headache using GASH’s poorly designed command-line interface, I built a macOS app using Swift 4.1, which talks to GASH through a new API in C.
